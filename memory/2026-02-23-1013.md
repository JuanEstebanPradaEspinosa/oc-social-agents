### ✉️ Advanced Options
- **Attachments**:  
  ```bash
  gog gmail send --to ... --body "See attachment" --body-file ./file.pdf
  ```
- **Track opens** (requires Cloudflare Worker setup):  
  ```bash
  gog gmail send --to ... --track --body-html "<p>Hi!</p>"
  ```
- **Reply with quoted original**:  
  ```bash
  gog gmail send --reply-to-message-id <ID> --quote --to ...
  ```

### 📁 Draft Management
```bash
gog gmail drafts list
gog gmail drafts create --subject "Draft" --body "Content"
gog gmail drafts update <draftId> --body "Updated"
gog gmail drafts send <draftId>
```

### 🔍 Email Operations
```bash
gog gmail labels list
gog gmail labels create "Label Name"
gog gmail labels modify --add STARRED --remove INBOX
gog gmail batch delete --ids <msgId>
```
